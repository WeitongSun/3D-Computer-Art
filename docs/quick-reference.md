# Quick Reference & Useful Links
# 快速参考与实用链接

## Quick Reference · 快速参考

| Task · 任务                       | Where / What · 位置 / 做什么                                     |
|-----------------------------------|------------------------------------------------------------------|
| Download Unity Hub · 下载 Unity Hub | [unity.com/download](https://unity.com/download)                  |
| Unity 2022 LTS page · Unity 2022 LTS 页面 | [unity.com/releases/2022-lts](https://unity.com/releases/2022-lts) |
| Meta XR All-in-One SDK            | [assetstore.unity.com](https://assetstore.unity.com/) (search: *Meta XR All-in-One SDK*) |
| Meta Developer Portal · Meta 开发者平台 | [developer.oculus.com](https://developer.oculus.com/)             |
| Download SideQuest · 下载 SideQuest | [sidequestvr.com/setup-howto](https://sidequestvr.com/setup-howto) |
| Enable Developer Mode · 启用开发者模式 | Meta Horizon app → Devices → your Quest → Developer Mode → ON     |
| Build for Quest · 为 Quest 构建    | `File → Build Settings → Android → Build`                         |
| Sideload APK · 侧载 APK            | SideQuest → *Install APK from folder* → select your `.apk`        |
| Find your app on headset · 在头显中找到你的应用 | `App Library → filter by Unknown Sources`                         |
